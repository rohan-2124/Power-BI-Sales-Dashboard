# 📊 Power BI Sales Dashboard

I created this Sales Dashboard in Power BI to analyze sales, profit, customers, and product performance. The dashboard provides a quick overview of business performance using interactive charts and filters.

### Features

* Total Sales, Profit, and Quantity KPIs
* Monthly Sales Trend Analysis
* Profit by Product Sub-Category
* Sales by State
* Sales by Customer
* Payment Mode Analysis
* Category-wise Quantity Analysis
* Interactive Filters and Slicers

### Tools Used

Power BI | DAX | Power Query | Data Modeling | Data Visualization

This dashboard helps users track sales performance, identify top customers and products, and make better business decisions using data.
